    description: Dimensionamos la potencia a instalar en base a los datos obtenidos del analisis.

  - title: Instalación y puesta en marcha
    description: Instalamos el sistema propuesto y comienza una nueva etapa.
    image: "/images/how-it-work/4.webp"

  - title: Servicio Post-Venta
    description: Realizamos mantenimientos anuales del sistema instalado. Para garantizar el buen funcionamiento del mismo.
    image: "/images/how-it-work/1.webp"
